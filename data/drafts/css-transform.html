Dealing with CSS Transforms in Javascript

CSS TRansforms

Posted on ...

Transform is a superior form of CSS property. It is fast, powerful and magic. It also has a nasty syntax and some browser only recognize it under it's prefixed form, so whenever you try to use it in the code it quickly turns into a mess.

This article will show you how to create a simple wrapper around the CSS transform property in Javascript. This wrapper can be used to move and animate HTML elements using CSS transform while keeping the code nice, tiday and very readable.
