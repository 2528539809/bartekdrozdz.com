5 different sound visualization effects.

Use the options menu on the opt left to change effects and/or switch to microphone input. Track: "Hide or Seek" by SBTRKT. Full source code for this demo is available on Github.

Technology: WebGL, WebRTC, Web Audio API

You need Chrome to run this experiment. It requires WebGL, WebRTC and the Web Audio API to function properly.
