5 different sound visualization effects + Leap Motion interactivity

Use the options menu on the top left to change effects and toggle between the track and microphone input. If you have a LeapMotion device you can use it to interact with the visualizations.

You need Chrome to run this experiment. It requires WebGL, WebRTC and the Web Audio API to function properly.

Technology: WebGL, WebRTC, Web Audio API

This is a serie of visuals I created for a party organized by Tool in February. They were projected on a wall and guests could interact with them using a Leap Motion device.

Track: "S.P.Y." Codex Machine
