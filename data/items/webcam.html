Bad reception effect of a webcam feed.

Playing with some shader to create a bad recetion effect for you camera feed. Mouse your mouse around (or tap anywhere on mobile device) to change the effect and it's intensity.

Note: Be sure to click Allow on the toolbar above.

Technology: WebGL, WebRCT

You need a WebGL and WebRCT enabled browser run this experiment. Chrome on desktop and Android is currently the only browser to support those both APIs.
