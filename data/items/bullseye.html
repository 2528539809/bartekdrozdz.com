A mobile game experience.

Bullseye’s Playground is a collection of 3D games filled with holiday spirit. A magical collection of characters led by Target’s lovable mascot Bullseye populate a world of holiday joy. The project was made in partnership between Target and Google’s Art, Copy & Code team. Art, Copy & Code is a Google initiative created to explore new ways in which technology can help build brands in a digital world.

Technology: HTML5, WebGL, mobile

  • Role: Developer
  • Company: Psyop, 72&Sunny
  • Release date: December 2014
  • Technologies: WebGL, HTML5, Javascript
You need a browser that supports the canvas element to run this experiment. Chrome, Firefox, Safari or IE 9 and up will do.
